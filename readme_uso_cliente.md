# 📘 GUÍA DE USO – NEXORA RENT

---

## 🏠 Flujo básico

### 1. Crear propiedad

Menú → Propiedades → Nueva

---

### 2. Añadir reservas (vacacional)

Menú → Reservas → Nueva reserva

---

### 3. Crear contrato (larga duración)

Menú → Alquileres → Nuevo contrato

⚠ No se permite si existen reservas en esas fechas

---

### 4. Gestión económica

* ingresos automáticos
* control de gastos
* seguimiento financiero

---

### 5. Tareas

* limpieza
* mantenimiento
* control operativo

---

### 6. Informes

* rentabilidad por propiedad
* vacacional vs larga duración

---

## 🔁 Compatibilidad del sistema

Una propiedad NO puede tener:

* reservas vacacionales
* contrato de larga duración

en las mismas fechas

---

## 🔐 Licencias

Si la licencia expira:

* puedes acceder
* puedes consultar
* no puedes operar

👉 No se pierde información

---

## 📧 Correo

Configurable en:

Perfil → Configuración de correo
